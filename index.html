Features

Everything you need for document discovery

Find and download files from multiple government agencies, filter by document type, and build a local archive of public records.

Multi-Source Discovery

Find documents from 27 government agencies simultaneously from a single interface.

Auto-Extract Archives

ZIP files automatically extracted after download with progress tracking.

Library Viewer

Gallery view of extracted PDFs, videos, and audio with thumbnail generation.

Persistent Queue

Downloads persist across page navigation. Start from Archives, monitor from Queue.

Built-in PDF Viewer

Syncfusion PDF viewer with search, zoom, and annotation support.

Media Player

Integrated media_kit player for MP3, MP4, WAV, and MOV files.

Torrent Support

aria2c integration for large datasets via magnet links.

Privacy First

DuckDuckGo search by default. No accounts, no tracking, no cloud.

Multi-Agency Discovery

Find documents from FBI Vault, DOJ archives, Federal Court records, CIA FOIA Reading Room, and 27 other government agencies from a single unified interface. Advanced filtering by institution category, time range, and file type. Downloads files—does not search within document contents.

Advanced Search

Build precise queries with quick tags, date range filters, and file type selection. Choose your preferred search engine and institution categories.

  • Quick tags for common search terms
  • Time range filters (last month, year, all time)
  • 27 file type filters (PDF, DOC, XLS, media)

Search Results

Browse results with source URLs, file sizes, and direct download links. Add items to the queue or download immediately.

Persistent Download Queue

Download multiple documents simultaneously with real-time progress tracking. Downloads persist across page navigation—start a download from Archives and monitor it from Queue. Configure up to 10 concurrent downloads with automatic retry.

Document Library

Browse all extracted files in a gallery view with PDF thumbnail generation. Filter by dataset, file type, or search by filename. Built-in Syncfusion PDF viewer and integrated media player for audio/video files.

  • Grid and list view modes with sorting options
  • Auto-generated PDF thumbnails
  • Filter by dataset or file type

Fully Customizable

Tailor the app to your research workflow. Configure default search terms, preferred file types, download behavior, and storage locations.

  • Up to 10 concurrent downloads with auto-retry
  • Custom download location and file type filters
  • Automated snapshot scheduling

Technical Specs

Built for researchers

A desktop application built with Flutter for macOS, Windows, and Linux.

Cross-Platform

Native desktop app for macOS, Windows, and Linux. Built with Flutter for consistent experience.

Local Storage

Track your downloads and preferences locally. No cloud required, all data stays on your machine.

Privacy First

Uses DuckDuckGo for privacy-focused searching. No tracking, no accounts, no data collection.

PDF Viewer

Built-in Syncfusion PDF viewer with annotations. View documents without leaving the app.

Media Player

Integrated media_kit player for audio and video files. Play MP3, MP4, WAV, and MOV directly.

Open Source

MIT licensed, fully open source. Inspect the code, contribute, or fork for your own use.
